# CodeAlpha_Stock_Portfolio_Tracker 📈

## Description
A Python-based Stock Portfolio Tracker developed for the **CodeAlpha Python Programming Internship** (Task 2).

## Key Features
- **Dictionary Management:** Stores stock symbols and prices.
- **Dynamic Calculation:** Computes individual and total portfolio values.
- **File Handling:** Automatically exports results to a `.txt` report.

## Technical Stack
- **Language:** Python
- **Core Concepts:** Dictionaries, Input Validation, File I/O.
